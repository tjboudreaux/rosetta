The current primary datastore for the user-profile service (Project Nimbus) is Cloud Spanner, deployed as the unified ledger tier. The migration from the prior document store (DynamoDB) to Cloud Spanner was adopted to provide strong consistency for the service. This change is documented in the project history as "Project Nimbus migrates its primary store to the unified ledger tier for strong consistency, replacing the prior document store," with the tier glossary defining "unified ledger tier" as the Cloud Spanner deployment.
