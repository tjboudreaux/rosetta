The current primary datastore for the user-profile service is **Cloud Spanner**, as established in ADR 0651 ("user-profile service primary datastore: Cloud Spanner", accepted 2026-06-08). The service (Project Nimbus) migrated from DynamoDB to Cloud Spanner as part of the "unified ledger tier" to achieve strong consistency. ADR 0651 designates Cloud Spanner as the current standard for all new user-profile code.
