The user-profile service's primary datastore is Cloud Spanner (ledger tier, cluster spanner-prod-eu), as established in ADR 0278 dated 2026-06-09. This decision supersedes earlier implementations using DynamoDB and Postgres, with the Atlas team migrating to Cloud Spanner for strong consistency requirements. The legacy DynamoDB configuration in profile_store.py is stale and should not be treated as current.
