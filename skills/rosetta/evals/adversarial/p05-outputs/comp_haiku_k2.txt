The current primary datastore for the user-profile service is Cloud Spanner, specifically the "ledger tier" cluster spanner-prod-eu. This decision is documented in ADR 0278 (accepted 2026-06-09), which supersedes prior decisions to use DynamoDB (ADR 0276) and Postgres (ADR 0277). The migration to Cloud Spanner was adopted for strong consistency requirements, and any legacy references to DynamoDB in the codebase should be disregarded as they are marked stale/mid-migration.
