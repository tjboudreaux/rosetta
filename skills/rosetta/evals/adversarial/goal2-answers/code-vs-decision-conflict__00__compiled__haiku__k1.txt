Beacon (notification dispatcher) is running on MySQL in production. The production code (notify_svc.py contains DATASTORE = "mysql") and git commit 55dd66e (2026-05-24, "notify_svc: provision MySQL (prod), retire Postgres POC") confirm this is the shipped reality. ADR 0001 originally claimed Postgres (Status: Accepted, 2026-02-01) but is now marked Superseded by ADR 0002 because a licensing blocker prevented the Postgres POC from reaching production. ADR 0002 (Status: Accepted, 2026-05-25) correctly documents MySQL as the actual datastore and supersedes the stale ADR 0001 record. The divergence occurred because ADR 0001 was never amended after the licensing blocker forced the team to pivot mid-build.
